## NozyIO

**Python AI/ML pipeline orchestration UI**

- Python pipeline orchestration and visual scripting
- Convert and export the workflow json as Python code

Start web dev server:
`cd web && npm run dev`

Start python server:
`python -m nozyio.server --allow-cors`
